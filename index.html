Tentang Saya

Hi.. Perkenalkan nama saya Muhammad Rifki Nurul Ramdani Alamsyah. Nama panggilan saya Rifki. saya lahir di Sukabumi pada bulan november tahun 2000. sejak kecil, saya adalah orang yang menyukai teknologi itu di buktikan saya umur 5 tahun sudah mahir dalam menggunakan Hanphone Nokia jadul ya walaupun hanya sekedar bermain game. di umur 7-10 tahun saya bermain Playstation 2 tentunya tidak hanya itu saya tertarik menganalisa komponen nya penyebab rusak salah satu nya yaitu optik ps2 nya. Nah menginjak umur 12-15 Tahun saya tertarik dalam perakitan komputer di umur tersebut saya sangat menyukai mempelajari bagaimana komponen komputer mempengaruhi kecepatan kinerja komputer dalam bermain Game, contohnya ya kalau ingin main Game ini komputer kita spesifikasi nya harus bagaimana dan komponen apa saja yang harus di penuhi agar kita lancar bermain game tersebut.

Setelah memasuki SMA, tepatnya di umur 16 tahun saya mulai tertarik dalam pembuatan aplikasi ya seperti coding namun pada massa SMA saya pun masih senang bermain game jadi pada massa itu saya tidak terlalu banyak memperlajari pembuatan aplikasi, itu adalah penyesalan saya menghabiskan waktu karena game. namun hikmah di balik semua ini kemampuan bahasa inggris saya terasah dan menjadi lebih baik itu di karenakan game umum nya story nya menggunakan bahasa inggris

Memasuki dunia Perkuliahan baru lah saya mulai serius mempelajari pengembangan aplikasi lebih dalam.menjadi mahasiswa sistem informasi di Universitas Nusa Putra Sukabumi, konsentrasi pada rekayasa Perangkat Lunak. Saya senang belajar lebih banyak tentang pemrograman web, Basis Data, analis Sistem, dan front end development. Saya juga seorang mahasiswa yang aktif bekerja di organisasi di universitas saya. Saya suka menantang diri sendiri dan belajar hal-hal baru. Berkomitmen dan Konsisten pada suatu hal.
